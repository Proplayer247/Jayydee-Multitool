# 👑 Jayydee Multitool - Open Source Edition

Welcome to the official source code for **Jayydee's Multitool**. This tool was built to provide a safe, fast, and professional way to manage Roblox ban traces and system utilities.

## 🛡️ Why Open Source?
We believe in transparency. Some antivirus programs might flag this tool because it edits the Windows Registry (to clear Roblox traces). By providing the source code, you can see exactly what happens under the hood:
- **No Hidden Malware**: Pure C# logic.
- **No Data Stealing**: The only internet connection is for auto-updates and Discord status.
- **Native Performance**: No slow batch scripts.

## 🚀 Key Features:
- **Roblox Trace Purge**: Deletes deep registry keys and hidden folders that Roblox uses to track bans.
- **TMAC Spoofer**: Quick access to Technitium MAC Address Changer.
- **Windows Activation**: Integrated MAS engine for genuine Windows status.
- **Discord Rich Presence**: Show your friends what you're using.
- **Auto-Updater**: Always stay on the latest version.
